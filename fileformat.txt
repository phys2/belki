A ZIP file to combine data, clustering, dimred results.

Supported contents (no. of entries on squared brackets):

belki-<version>  // empty file to annotate zip [1]
input/<name>.tsv // a peakvolumes dataset [1]
input/<name>/<hash>.sha256 // hash of above file [0-1] (if missing or mismatch, data may be re-calculated and hash added)
	Note: right now the hash entry is mandatory
input/<name>/displays/<type>.tsv // a reduced dataset [*]
annotations/<aname>.tsv // uniprot annotations or flat clustering [*]
markerlists/<mname>.txt // lists of marked proteins [*]
hierarchies/<hname>.<?|json> // hierarchical clustering result [*]
[partitions/<pname>.? // labeled flat clustering]
